# Helping information on sphinx

## Update Documentation:
1. Make changes to the content files in the sphinx folder (.rst files)

2. cd to the sphinx folder:

```
cd sphinx
```

3. Build the Project

```
sphinx-build -b html . ./_build
```

4. Copy all files in .sphinx/_build/html/ to .docs/
5. Push your changes to github

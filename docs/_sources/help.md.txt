# Helping information on sphinx

## Update Documentation:
1. Make changes to the content files in the sphinx folder (.rst files)

2. Build the Project

```
sphinx-build -b html sphinx sphinx/_build
```

3. Copy all files in .sphinx/_build/html/ to .docs/
4. Push your changes to github

class invert.solvers.base.BaseSolver(regularisation_method='GCV', n_reg_params=24, prep_leadfield=True, use_last_alpha=False, verbose=0)#
Parameters

  • regularisation_method (str) –

    Can be either

    ”GCV” -> generalized cross validation “L” -> L-Curve method using triangle method “L_new” -> L-Curve method using triangle and scale-free params “Product” -> Minimal product method

  • n_reg_params (int) – The number of regularisation parameters to try. The higher, the more accurate the regularisation and the slower the computations.

  • prep_leadfield (bool) – If True -> Apply common average referencing on the leadfield columns.
